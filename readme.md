# For the Love of Film

A Front-End app that helps locate new and interesting movies and T.V. shows to watch.

Simple design that takes in user input and searches the TV Maze API for related shows.

## Run Locally

Clone the project

```bash
  git clone https://github.com/Team-Cache-Out/tech-port
```

Go to the project directory

```bash
  cd techport
```

Install dependencies

```bash
  npm install
```

Build the database
```bash
  psql postgres
  \i db/migration.sql
```

Start the server

```bash
  npm node server
```

Start the application
```bash
  npm startdev
```
